# Semantic File Searcher

## 개요
Windows 용 파일 검색 데스크탑 앱. 파일명 검색 뿐 아니라 파일 내용도 자연어로 검색 할 수 있도록 함.

## 기능
### 1) 검색 기능
1) 단순 파일명 / 폴더명 통합 검색
2) 확장자 필터링
3) 내용 기반 검색 
    - 문서 (txt, pdf, MS Office 문서 파일 형식)
    - 이미지 파일
4) 태그 기반 검색/필터링

### 2) 인덱싱 관리 기능
기본 사용자 폴더(다운로드, 문서, 사진, 동영상) 들은 디폴트로 인덱싱 대상.
1) 인덱싱에 추가 및 삭제할 폴더 / 파일 수동 선택 기능
2) 인덱싱 대상으로 등록된 폴더 내 변동 사항 감지 시 자동으로 인덱싱 업데이트
3) 태그 기반 관리 기능
    - 파일에 태그를 추가/삭제/수정 할 수 있어야 함.
    - 파일과 별도로 태그 목록을 관리 할 수 있어야 함.(태그 추가/삭제/수정)
    - 태그는 여러 개 추가 가능.
    - LLM / VLM 연동하면 감지된 신규파일에 대해 자동으로 태그를 생성.(태그 생성시 이미 등록된 태그를 우선적으로 등록하고, 적절한 기존 태그가 없으면 신규 태그 생성)

## 시스템 요구사항
1) 최소 요구사항
    - RAM 16GB 이상
2) 권장 요구사항
    - RAM 32GB 이상
    - CUDA 지원되는 GPU, VRAM 8GB 이상

## 제약 사항
모든 기능은 온라인 연결 필요 없이 온 디바이스로 동작해야 함.(적어도 설치가 다 되어있는 상태에서는 추가적인 인터넷 연결이 필요없어야 함.)
