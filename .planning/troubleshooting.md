## 기능 추가/개선

### 태그 기능 개선
1) 태그 자동생성 기능 외부화
    - 현재 설정메뉴 UI만 구현되어있고 연결된 백엔드 로직이 없거나 동작하지 않는 상태로 보임. 저장해도 config.json에 변화 없고, 설정 저장 후 신규 파일 감지시 태그 생성 적용도 안되는 것으로 보임.
2) 태그 관리 기능 구현
    - [x] 검색결과 목록에서 점 세개 버튼 눌러서 '태그 관리' 메뉴를 클릭해도 아무 반응 없음.
    - [x] '파일 - 설정 - 태그 관리' 메뉴에 들어가면 태그를 추가/수정하거나 삭제하는 기능에 대한 UI 버튼도 미구현.
