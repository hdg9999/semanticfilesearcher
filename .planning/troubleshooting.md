* 조치된 내용은 체크박스를 추가하여 마킹할 것. 대괄호 안에 확인 필요라고 되어있으면 기존에 조치했으나 여전히 문제가 있는 경우라고 사용자가 
확인한 내용임.


## 기능 추가/개선


### UI/UX 개선
1) 보기모드 선택 버튼 개선
    - 목록형 / 아이콘형 두개의 보기 모드 변경 버튼을 아이콘으로 변경
    - 배치를 검색 모드 드롭다운 아래, 검색 결과창 위 좌측 상단에 배치
    - 어떤 모드인지 직관적으로 알 수 있는 디자인의 아이콘 이미지는 직접 생성해서 적용.

2) 대기열 버튼 / 팝업 개선
    - 대기열 혹은 '큐'를 직관적으로 이해하기 쉬운 아이콘 이미지를 생성해서 해당 아이콘을 버튼으로 교체.
    - 대기열에 몇개 있는지는 모바일 앱의 알림뱃지 처럼 버튼 우측 상단에 숫자로 표시.
    - 대기열 버튼 시 나타나는 팝업에는 제목 표시줄 제거.(닫기버튼[X] 만 보이도록 개선)
